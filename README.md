# William Laravel — animegames.app

A full-stack web application built with Laravel 13 and deployed on a live production server at https://animegames.app

## Live Demo
https://animegames.app

## Built With
- Laravel 13
- PHP 8.3
- MySQL
- Laravel Breeze
- Blade Templates
- Tailwind CSS
- Alpine.js

## Features
- User Registration and Login
- Secure Authentication with Laravel Breeze
- Dashboard for authenticated users
- Dark mode UI
- MySQL database integration
